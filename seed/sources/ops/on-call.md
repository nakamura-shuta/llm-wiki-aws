# オンコール体制

## 概要

夜間（22:00-06:00 JST）および休日は、オンコール担当の agent が一次受けを行う。

## ローテーション

- IT 部門: 週替わり、agent 6 名で回す
- 総務・人事: オンコール対象外（翌営業日対応）

## オンコール中の対応範囲

| 優先度 | 夜間・休日の対応 |
|---|---|
| P1 | 即時対応（夜間運用上の内部目標: 30 分以内に初回応答。正式な SLA は `domain/sla.md` を参照） |
| P2 | 翌営業日の朝イチで対応 |
| P3 | 翌営業日の通常キューに入る |

## PagerDuty 連携

- P1 チケットが起票されると PagerDuty で on-call agent に通知
- 5 分間応答なしで次の on-call agent にフォールバック
- PagerDuty schedule は毎月第 1 営業日に lead が更新

## 手当

- オンコール待機: 2,000 円/日
- 夜間実対応: 1 件あたり 5,000 円（P1 のみ）
- 人事部の勤怠システムと連携して自動集計（月末締め）
