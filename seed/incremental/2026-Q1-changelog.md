# Tickflow 2026-Q1 Changelog

## 2026-01-15: SLA 短縮（P1）

**変更**: P1 の初回応答 SLA を **1 時間 → 45 分**に短縮。

背景: 2025-Q4 の SLA 遵守率分析で、P1 の初回応答が平均 38 分だったため、目標を引き締めて品質水準を上げる。解決 SLA (24 時間) は据え置き。

影響範囲:
- `domain/sla.md` の SLA マトリクス
- SLA engine のタイマー設定（DB の `sla_configs` テーブル）
- ダッシュボードの警告閾値

## 2026-02-01: カテゴリ追加（IT-AI）

**変更**: L2 カテゴリに `IT-AI`（AI ツール関連）を新設。

背景: ChatGPT / Copilot 等の AI ツール導入に伴い、ライセンス管理・利用ポリシー・トラブルの問い合わせが月 30 件超に増加。既存の `IT-SW` に混在していたため分離。

影響範囲:
- `domain/category.md` のカテゴリ体系
- 自動カテゴリ推定モデルの再学習

## 2026-03-10: エスカレーション条件変更

**変更**: P1 の自動エスカレーション条件を「アサインから 30 分」→「アサインから 20 分」に短縮。

背景: 2026-01-15 の SLA 短縮（45 分）に合わせ、エスカレーションまでの猶予も縮小。

影響範囲:
- `ops/escalation.md` の自動エスカレーション条件
- エスカレーション engine の設定値
